# 🎮 Quiz de Gestão Documental (HTML)

Projeto simples de **gameficação em Gestão Documental**, desenvolvido apenas com **HTML e JavaScript básico**, com perguntas interativas onde o usuário pode escolher respostas e receber feedback imediato.

## 📌 Objetivo

Este projeto tem como objetivo:
- Facilitar o aprendizado sobre **Gestão Documental**
- Servir como **material didático**, treinamento ou demonstração
- Mostrar um exemplo simples de **interação no HTML** sem uso de frameworks

## 🧩 Funcionalidades

- ✅ Perguntas de múltipla escolha  
- ✅ Interação por clique  
- ✅ Feedback imediato (correto / incorreto)  
- ✅ Código simples e fácil de entender  
- ✅ Funciona offline em qualquer navegador  

## 🛠️ Tecnologias Utilizadas

- **HTML5**
- **JavaScript (vanilla)**

Não utiliza bibliotecas, frameworks ou dependências externas.

## 📂 Estrutura do Projeto

```
/
└── index.html
```

Todo o código está concentrado em um único arquivo para facilitar o uso e o aprendizado.

## ▶️ Como Executar

1. Faça o download ou clone este repositório:
   ```bash
   git clone https://github.com/seu-usuario/nome-do-repositorio.git
   ```

2. Abra o arquivo `index.html` em qualquer navegador (Chrome, Edge, Firefox, etc).
